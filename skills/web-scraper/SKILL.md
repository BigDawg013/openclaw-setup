# Web Scraper

Extract structured data from public websites with built-in security controls.

## Capabilities

- E-commerce product data extraction
- Real estate listing scraping
- Job listing aggregation
- Social media data collection
- Output formats: CSV, JSON, Excel

## Security

- SSRF protection
- Rate limiting
- URL validation
- Secure browser-based scraping (Puppeteer/Cheerio)
